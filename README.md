**Технарь** — это голосовой и визуальный помощник для пилотов Microsoft Flight Simulator (MSFS). Программа подключается к симулятору через SimConnect, анализирует параметры полёта и предоставляет полезную информацию, звуковые оповещения и автоматические отчёты.
## Возможности
- **Автоматическое определение полёта** – контроль активируется при движении >5 км/ч.
- **Информация о воздушном судне** – название, бортовой номер, топливо, высота аэродрома.
- **Высотные отметки** – звуковое оповещение при достижении 1000, 1500, 3000, 5000 метров.
- **RVSM** – проверка эшелонов (чётные/нечётные) с учётом курса (восток/запад) после 45 секунд стабильности.
- **Контроль топлива** – оповещение при порогах 75%, 50%, 35%, 25%, 10% (с округлением до целых и массой в кг).
- **Анализ посадок** – вертикальная скорость, оценка (от «Очень мягкая» до «Превышение ограничений»), звуковая шкала.
- **Графический лаунчер** – управление запуском/остановкой, лог событий, анимация пилота, индикатор топлива.
- **Расчёт TOD (Top of Descent)** – встроенный калькулятор точки начала снижения.
- **Регулировка громкости** – кнопка с тремя уровнями (100% / 50% / 0%) для всех звуков.
- **Автоматическое обновление** – программа проверяет новую версию на GitHub и предлагает скачать и установить.

## Установка и запуск

1. Скачайте последнюю версию `Technician.exe` из раздела [Releases](https://github.com/HuanIvanov/MSFS-Teh/releases).
2. Запустите файл – никаких дополнительных установок не требуется.
3. Убедитесь, что Microsoft Flight Simulator запущен.
4. В лаунчере нажмите кнопку «Пуск».
5. Программа автоматически подключится к симулятору.

## Обновление
При запуске программа проверяет наличие новой версии. Если обновление доступно, появится диалог с предложением скачать и установить. После подтверждения программа самостоятельно заменит себя и перезапустится.
## Системные требования

- Windows 10 / 11 (64-bit)
- Microsoft Flight Simulator (2020 или 2024)
- SimConnect (устанавливается вместе с MSFS)
